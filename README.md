```sh
Maven
```

## Getting Started
Clone o repositorio e executa com algumas dessas IDE
```sh
Spring Tool Suite
IntelliJ IDEA
```

ou abrindo terminal na pasta do projeto, rode o comando
```sh
./mvnw spring-boot:run
```
_Obs: necessita do maven na sua maquina_
